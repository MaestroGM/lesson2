## Списки

Чтобы добавить ненумированные списки, необходимо пункты выделить (*). Вот так :
* Элемент 1
* Элемент 2
* Элемент 3

Чтобы добавить нумированные списки, необходимо пункты просто пронумировать. Например вот так:
1. Пункт
2. Пункт
3. Пункт
